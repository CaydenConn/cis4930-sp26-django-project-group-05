# PROJECT 1: Analyzing Movie Success Factors

## Description

  For our project we have scraped a CSV file containing 7,668 movies released between 1980-2020 that consist of 15
  attributes such as budget, gross revenue, IMDb score, genre, rating, and country of production. The goal is to investigate which factors contribute to a movie's success. By cleaning and processing the data, we aim to uncover trends in movie ratings, revenue generation, profitability, and how these patterns vary by genre, rating, and country of origin.
  
## Research Questions

1) How had the average rating of R-rated movies changed from 1980-2020?
2) Does a certain genre generate high gross revenue?
3) How are IMDb scores distributed across all action movies?
4) Do higher budget movies generate more revenue?
5) Which countries produce the most profitable movies?
