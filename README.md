# Project 3: Web Development Project

## Description
This project is all about taking our previous projects and combining them into a web app. We took our weather API and Movie theater and created a full scale application with the data.

Features: Weather Records, Movie database plus additions, movie analytics and graphs

### Instructions:
- Clone the repository `git clone {link}`
- Run a virtual environment
- `pip install -r requirements.txt`
- `python manage.py migrate`
- `python manage.py seed_data`
- `python manage.py fetch_data`
- `python manage.py runserver`
